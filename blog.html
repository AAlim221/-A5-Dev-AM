Question 1: What are the different ways to select an element in the DOM?

1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

Question 2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets or sets the HTML markup inside an element.
innerText: Gets or sets the text considering CSS display styles.
textContent: Gets or sets the plain text inside an element.

Question 3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is used for a parent element to manage events from child elements, improving performance.

Question 4: What is event bubbling in the DOM?

Event bubbling refers to the event propagation in which an event moves from the deepest target element upwards to the parent elements.

Question 5: How do you create, add, and remove elements using JavaScript?

- Create: document.createElement()
- Add: appendChild() or insertBefore()
- Remove: removeChild() or remove()
